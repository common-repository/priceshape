=== Description ===

PriceShape Plugin is a tool getting relevant products price.
We primarily help webshops streamlining their pricing strategy based on competitor data, including: Prices, Stock and Assortment.
With PriceShape, you get a complete overview of competitors' prices through our intelligent dashboard with daily updates through price monitoring, improving decision-making related to pricing strategies.

=== Basic Features ===

1. Easy to install

2. User-friendly dashboard.

3. Apply PriceShape to products.

4. Select categories from dropdown.

5. Select approve or deny new product price action.

6. Update product prices.

7. You can update prices for specific products.

=== Automatic Installation ===

1. Go to your plugin browser inside your wordpress installation and search `PriceShape` by keyword. Then choose "PriceShape" and click install. It will be installed shortly.

2. Activate the plugin from `Plugins` menu after installation

=== Manual Installation ===

1. Download the latest version and extract the folder to the `/wp-content/plugins/` directory

2. The plugin will appear as inactive in your `Plugins` menu

3. Activate the plugin through the `Plugins` menu in WordPress
